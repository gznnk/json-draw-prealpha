CSSプロパティの順番を、外側→内側、レイアウト→装飾→フォントのようにグループごとに分類して並び変えて。

例：
.selector {
  display: flex;
  position: relative;
  top: 0;
  right: 0;
  width: 100%;
  height: 100%;
  margin: 0;
  padding: 0;
  border: 1px solid #ccc;
  background-color: #fff;
  font-size: 16px;
  font-weight: bold;
  color: #333;
  text-align: center;
  cursor: pointer;
  transition: all 0.3s;
}
